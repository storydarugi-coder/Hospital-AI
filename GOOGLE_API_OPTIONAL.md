# 📋 Google Custom Search API - 선택사항

## ✅ 현재 상태
- ✅ **네이버 크롤링 (1순위)**: 정상 작동 중
- ⚠️ **Google API (2순위)**: 미설정 (선택사항)
- ✅ **유사도 검사**: 네이버만으로도 충분히 작동

## 🎯 작동 방식

```
유사도 검사 시작
    ↓
1단계: 네이버 블로그 크롤링 (무료, 한국어 최적화)
    ↓
    ✅ 매칭 발견 → 결과 표시
    ↓
    ❌ 매칭 없음 → 2단계로
    ↓
2단계: Google Custom Search (선택사항, API 키 필요)
    ↓
    환경변수 없음 → 조용히 스킵
    ↓
결과: 네이버 결과만 표시 (정상 작동)
```

## 📊 네이버 vs Google

| 항목 | 네이버 크롤링 | Google API |
|------|--------------|-----------|
| **비용** | ✅ 무료 | 무료 100회/일 |
| **한국어** | ✅ 최적화 | 보통 |
| **최신 글** | ✅ 6개월 | 전체 기간 |
| **설정** | ✅ 불필요 | API 키 필요 |
| **추천** | ✅ 필수 | 선택사항 |

## 결론
**Google API 없어도 문제없습니다!** 
네이버 크롤링만으로 충분히 유사도 검사가 가능합니다.

---

## (선택) Google API 설정 방법

필요하시면 나중에 설정 가능:

1. Google Cloud Console에서 API 키 발급
2. Cloudflare 환경변수 추가:
   - GOOGLE_API_KEY
   - GOOGLE_SEARCH_ENGINE_ID

하지만 **지금은 설정 안 해도 됩니다!**
